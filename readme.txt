== Description ==

Web2SMS extends WooCommerce by adding SMS options on order status.

= Features: =

* **100% FREE TO USE** (GPLv2 license).
* Integrates Web2sms SMS service with your WordPress + WooCommerce online shop.

== Installation ==

1. Install the plugin through the WordPress plugins screen directly (recommended).

2. Activate the plugin through the `Plugins` menu in WordPress.

3. Configure your settings under `WooCommerce > Settings > SMS > Web2sms` option panel
